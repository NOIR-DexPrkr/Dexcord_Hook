# 🚀 Dexcord Hook

[English](README.md) | [Español](README.es.md)

**Dexcord Hook** es un panel de control moderno, premium y potente para la gestión de Webhooks de Discord. Diseñado con un enfoque en la velocidad, la estética y la experiencia de usuario, te permite gestionar múltiples webhooks, redactar mensajes complejos con embeds y realizar un seguimiento de tu historial, todo desde una interfaz hermosa y fluida.

## ✨ Características

- **🌐 Multi-idioma Bidireccional**: Totalmente localizado en inglés y español.
- **⚡ Redactor Avanzado**: Alterna entre un formulario intuitivo y un editor JSON puro con sincronización en tiempo real.
- **👁️ Vista Previa en Vivo**: Mira exactamente cómo se verá tu mensaje en Discord antes de enviarlo.
- **📦 Gestor de Webhooks**: Registra múltiples webhooks y cambia entre ellos sin esfuerzo.
- **📜 Historial Inteligente**: Mantén un registro de los mensajes enviados. Edítalos o elimínalos directamente desde el panel (¡se sincroniza con Discord!).
- **💎 Interfaz Premium**: Diseño moderno estilo glassmorphism con animaciones suaves y diseño responsivo.
- **💾 Persistencia Local**: Tus webhooks e historial se guardan de forma segura en el almacenamiento local de tu navegador.

## 🛠️ Tecnologías

- **Framework**: [React 19](https://react.dev/)
- **Herramienta de Construcción**: [Vite](https://vitejs.dev/)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: Vanilla CSS (Sistema de diseño personalizado con variables CSS)
- **Iconos**: Iconos SVG optimizados y personalizados

## 🚀 Primeros Pasos

### Requisitos Previos

- [Node.js](https://nodejs.org/) (Se recomienda la versión LTS más reciente)
- [npm](https://www.npmjs.com/)

### Instalación

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tuusuario/hookdex.git
   ```
2. **Navegar al directorio**:
   ```bash
   cd hookdex
   ```
3. **Instalar dependencias**:
   ```bash
   npm install
   ```
4. **Iniciar el servidor de desarrollo**:
   ```bash
   npm run dev
   ```

## 📖 Uso

1. **Agregar un Webhook**: Ve a la pestaña de **Webhooks** y pega la URL de tu Webhook de Discord.
2. **Redactar**: Usa el **Redactor** para escribir tu mensaje. Puedes añadir embeds, campos, imágenes y más.
3. **Enviar**: Haz clic en "Enviar Webhook" y verás aparecer el mensaje en tu canal de Discord.
4. **Gestionar**: Usa la pestaña de **Historial** para editar mensajes anteriores o eliminarlos de Discord.

## 🔒 Privacidad y Seguridad

Dexcord Hook funciona completamente en tu navegador. Tus URLs de Webhook y tu historial nunca salen de tu máquina; se almacenan localmente y solo se envían directamente a la API oficial de Discord.
